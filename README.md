# ThinkDB Static Website

A premium static landing website for ThinkDB, owned by Narula Cloud & AI Consulting.

## Files

- `index.html` — main static website
- `styles.css` — complete responsive styling
- `script.js` — mobile navigation and scroll reveal animation
- `assets/` — logo and founder photo
- `404.html` — GitHub Pages fallback
- `site.config.js` — contact/company configuration reference

## Local preview

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8080
```

Then open:

```text
http://localhost:8080
```

## Deploy to GitHub Pages

Option 1: Root deployment

1. Copy all files into your repository root.
2. Commit and push.
3. Go to GitHub repository → Settings → Pages.
4. Select branch `main` and folder `/root`.
5. Save.

Option 2: `/docs` deployment

1. Create a `docs` folder.
2. Copy all files into `docs/`.
3. Commit and push.
4. Go to Settings → Pages.
5. Select branch `main` and folder `/docs`.
6. Save.
